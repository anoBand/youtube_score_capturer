
# 🎼 YouTube 악보 PDF 추출기

---

## 🚩 주요 기능

- **유튜브 영상에서 악보 자동 추출** 및 **PDF로 저장**

압축 파일을 다운로드한 후 원하는 위치에 압축을 풀어주세요.

## 실행 예시




* 추출 영역(%) 지정 시 그림판 좌하단의 커서 px 모니터링 활용 시 편리함  
* 파일 경로에 url 영상파일, 병합된 악보 PDF파일 저장됨  
* temp_images 파일은 임시 저장용 폴더이므로, 사용하지 않으면 됩니다.
