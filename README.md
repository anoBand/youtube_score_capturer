# yt-tab-img-extractor

Python 언어 사용

Korean comment supported. 한국어 주석 있음

***
## 추가 예정 기능

#### 병합 이미지를 PDF로 변환해 다운

#### 실행마다 영상 제목으로 폴더 생성 후 내부에 영상 파일, pdf파일, 이미지 파일 저장

#### 다른 악기의 악보 형식에도 적용

#### 일정 속도로 움직이는 악보 형식에 대해 추출  

#### 웹 UI로 구현해 따로 코드 파일 실행하지 않고도 기능 사용
